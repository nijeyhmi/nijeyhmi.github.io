# 개인 블로그 'Hey, Jeanz'

## 🔧 프로젝트 개요

- **프로젝트 이름**: 개인 개발 블로그
- **기술 스택**: React18, Tailwind CSS, GitHub Pages
- **목적**: 개발 관련 글을 Markdown 형식으로 작성하고 GitHub Pages를 통해 배포하는 블로그 제작

---

## ✅ 주요 기능 리스트

### 1️⃣ **포트폴리오 메뉴**

**설명**: 개발자가 진행한 프로젝트와 작업 내용을 정리하여 포트폴리오 페이지에 표시합니다.

#### 구현 세부사항:

- 프로젝트 카드 스타일로 목록을 표시
- 각 프로젝트 카드에 제목, 설명, 사용 기술, GitHub 링크 포함
- 클릭 시 상세 페이지로 이동 (React Router 사용)

#### 구현 방법:

- 컴포넌트: `PortfolioPage`, `ProjectCard`
- 사용 라이브러리: `React Router`, `react-icons`

#### 추가 기능:

- 프로젝트 필터 기능 (기술 스택별 필터)
- 외부 링크 아이콘 추가 (GitHub, Live Demo 등)

### 2️⃣ **카테고리 리스트 메뉴**

**설명**: 블로그 포스트를 주제별로 분류하여 카테고리별로 쉽게 접근할 수 있도록 합니다.

#### 구현 세부사항:

- 카테고리 목록을 사이드바에 표시
- 각 카테고리 클릭 시 해당 카테고리의 글 목록을 표시
- 카테고리별 포스트 수 표시

#### 구현 방법:

- 컴포넌트: `PostList`, `PostItem`, `Portfolio`
- 데이터: 카테고리 데이터는 JSON 또는 Firebase에서 가져오기
- 사용 라이브러리: `React Router`

#### 추가 기능:

- 동적 카테고리 생성 기능
- 카테고리별로 최신 글 정렬

### 3️⃣ **다크 모드 / 라이트 모드 토글**

**설명**: 사용자 환경에 맞게 블로그의 테마를 다크 모드와 라이트 모드로 전환할 수 있도록 지원합니다.

#### 구현 세부사항:

- 다크 모드 / 라이트 모드 전환 버튼을 헤더에 배치
- 전환 상태는 Local Storage에 저장하여 새로고침 시에도 유지

#### 구현 방법:

- 컴포넌트: `DarkModeToggle`
- 상태 관리: `useState`, `useEffect`

### 4️⃣ **Markdown 포스트 작성 및 렌더링**

**설명**: 블로그 포스트를 Markdown 형식으로 작성하고, 이를 HTML로 렌더링합니다.

#### 구현 세부사항:

- 포스트 목록 페이지 및 상세 페이지 구현
- 코드 블럭 하이라이트 기능 지원
- 이미지, 링크 등 Markdown 요소 지원

#### 구현 방법:

- 라이브러리: `react-markdown`, `remark-gfm`, `react-syntax-highlighter`

### 5️⃣ **검색 기능**

**설명**: 블로그 포스트를 제목이나 내용으로 검색할 수 있는 기능을 제공합니다.

#### 구현 세부사항:

- 검색창을 헤더에 배치하여 사용자가 빠르게 검색 가능
- 입력한 키워드로 포스트 제목 및 내용을 필터링
- 실시간 검색 결과 표시

#### 구현 방법:

- 컴포넌트: `SearchBar`
- 상태 관리: `useState`
- 데이터 필터링: `filter()` 메서드 사용

### 6️⃣ **페이지 배포**

**설명**: 블로그를 GitHub Pages를 통해 배포합니다.

#### 구현 세부사항:

- GitHub Actions 설정 파일 작성
- `gh-pages` 브랜치에 자동 배포 설정

#### 구현 방법:

- 라이브러리: `gh-pages`
- GitHub Actions 설정

### 7️⃣ **기본 이미지 자동 추가 기능**

**설명**: 포스트에 이미지가 없는 경우, 자동으로 기본 이미지를 추가하여 블로그의 일관된 디자인을 유지합니다. 각 포스트의 이미지는 기본적으로 작은 크기로 표시되어 텍스트가 강조되도록 합니다.

#### 구현 세부사항:

- 포스트 생성 시 이미지가 없는 경우, 기본 이미지 경로를 자동으로 설정
- 기본 이미지는 테마에 따라 다르게 적용 (예: 다크 모드/라이트 모드에 따라 다른 이미지)
- 이미지가 있는 포스트의 경우에도 크기를 작게 설정하여 텍스트가 강조되도록 조정

#### 구현 방법:

- 컴포넌트: `PostCard`, `DefaultImage`
- 상태 관리: 이미지 여부 확인 후 기본 이미지 설정
- 스타일링: CSS 또는 Tailwind CSS를 사용하여 이미지 크기 조정
