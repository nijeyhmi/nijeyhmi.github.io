React가 UI를 업데이트하는 법

Virtual DOM을 이용한 성능 향상

→ 그러나 자식 컴포넌트가 상태 변경의 영향을 받지 않을 때에도 리렌더링되는 문제 발생

기본적으로 App 구성 요소의 상태가 변경되면 이 변경에 직접적인 영향을 받지 않더라도 자식 구성 요소가 리렌더링된다

React 성능 최적화

필요한 경우 state를 local로 유지

컴포넌트 리렌더링이 필요한 경우에만 일어나도록 컴포넌트의 state를 담당하는 코드를 추출하여 이를 로컬로 만든다

React.memo()

memoization을 통해 동일한 결과를 재사용할 때 캐싱된 결과를 반환

동적 import() 사용

const Home = React.lazy(() => import("./components/Home"));
const About = React.lazy(() => import("./components/About"));
